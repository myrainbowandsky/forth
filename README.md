# 内容工厂Agent - 页面原型

## 项目概述

这是一个基于 Next.js 的内容工厂Agent原型系统，包含选题分析、内容创作和发布管理三大核心功能模块。

## 功能模块

### 1. 选题分析 (`/analysis`)
- 关键词搜索和分析
- 公众号文章数据分析
- AI生成洞察报告
- 高频词云展示
- 点赞TOP5和互动率TOP5文章展示

### 2. 内容创作 (`/create`)
- 基于洞察报告选题
- 自定义选题输入
- AI智能生成文章
- 自动插入Unsplash图片
- 多种写作风格选择
- 实时预览生成内容

### 3. 发布管理 (`/publish`)
- 文章列表管理
- 多状态管理（草稿、待审核、已发布）
- 一键发布到小红书/公众号
- 批量操作支持
- 数据统计展示

### 4. 仪表盘 (`/`)
- 数据统计概览
- 内容生产趋势图
- 平台发布分布
- 最新文章列表
- 热门话题展示

### 5. 设置 (`/settings`)
- AI模型配置
- 平台API配置
- 数据源设置
- 通知设置
- 安全设置

## 技术栈

- **框架**: Next.js 16 (App Router)
- **样式**: Tailwind CSS
- **图表**: Recharts
- **图标**: Lucide React
- **UI组件**: Radix UI

## 页面导航

访问首页后，您可以通过左侧导航栏访问不同功能模块：

1. **仪表盘**: 查看整体数据概览
2. **选题分析**: 输入关键词进行分析（点击"开始分析"查看模拟效果）
3. **内容创作**: 选择选题并生成文章（点击"开始创作"查看模拟效果）
4. **发布管理**: 管理所有文章和发布状态
5. **设置**: 配置API和系统参数

## 交互说明

当前为前端原型，所有数据为模拟数据：

- 选题分析页面点击"开始分析"会展示模拟的分析结果
- 内容创作页面点击"开始创作"会展示模拟的AI生成文章
- 发布管理页面可以查看模拟的文章列表和状态
- 设置页面可以填写配置但不会实际保存
